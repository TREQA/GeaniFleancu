apply-for-cc-marketing service

Is eligible

  • GET /v1/isEligible
  • Mandatory header: X-USER-CIF
  • Example:
{
    "CreditCard": {
        "eligible": true,
        "maxAmount": 120000,
        "minAmount": 700
    },
    "FlexiCredit": {
        "eligible": true,
        "amount": 120000,
        "duration": 60,
        "interest": 15,
        "interestType": "Fixa",
        "installmentValue": 900
    }
}

Apply for campaign

  • POST /v1/applyForCampaign
  • Mandatory headers: X-USER-CIF, Content-Type
  • Request body (cardType value is populated/chosen by FE team)
{
    "campaignName": "CreditCard",
    "campaignData": {
        "amount": 2000,
        "cardType": "Visa premium"
    }
}

{
    "campaignName": "FlexiCredit"
}

Custom notification events

Notification events

Custom notification levels are the same as participating levels. With custom notification levels you will also receive notifications for select events. To find out more, check out notification emails.
